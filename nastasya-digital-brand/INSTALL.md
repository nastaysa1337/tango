# Как открыть сайт

Проект лежит в папке `nastasya-digital-brand`.

```bash
cd nastasya-digital-brand
npm install
npm run dev
```

Открыть локально:

```bash
http://localhost:3000
```

Что уже залито:

- README
- package.json
- next-env.d.ts
- next.config.mjs
- postcss.config.js
- tailwind.config.js
- app/layout.tsx
- app/globals.css
- app/page.tsx

Важно: новая репа через доступный GitHub-инструмент не создаётся, поэтому сайт загружен отдельной папкой в `nastaysa1337/tango`, не ломая старый сайт в корне.
